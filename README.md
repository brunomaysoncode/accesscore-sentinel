## Pitch (30s)
Sistema visual premium para simular controle de acesso corporativo: cadastros, permissões, catracas, terminais, redes, coletor e relatórios. Ideal para apresentação comercial e validação de produto.

## Roteiro de demonstração
1. **Dashboard** — métricas e alertas.
2. **Cadastre um visitante** — gera código, valida e define catraca.
3. **Coletor** — simula leitura (entrada/saída), libera/nega e registra no histórico.
4. **Relatórios** — exporta CSV/JSON.

## Módulos
- Dashboard com cards + gráfico
- Visitantes / Empresas / Funcionários / Veículos
- Códigos de Acesso + validação
- Permissões (RBAC simulado)
- Catracas + Terminais (80) + Redes
- Entrada/Saída + histórico
- Relatórios + exportação

## Importante (Modo Demonstração)
Este protótipo roda em **HTML/CSS/JavaScript puro**, salva em **localStorage** e **não substitui** integração real com catracas/coletores, backend, banco, autenticação, criptografia e logs auditáveis.
